# Divisibility-Rule-of-14
So a number is divisible by 14 if it is divisible by both 2 and 7.  Conditions 1) Divisible by 2  Last digit must be even → 0, 2, 4, 6, 8  2) Divisible by 7  Use the rule of 7: 1) Take last digit 2) Double it 3) Subtract from remaining number 4) Repeat 5) If result divisible by 7 → original number divisible by 7
